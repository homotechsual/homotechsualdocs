## SYNOPSIS

Sends a request to the Halo API.

## SYNTAX

```powershell
Invoke-HaloRequest [[-WebRequestParams] <Hashtable>] [-RawResult] [<CommonParameters>]
```

## DESCRIPTION

Wrapper function to send web requests to the Halo API.
