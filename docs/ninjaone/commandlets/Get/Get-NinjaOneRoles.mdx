## SYNOPSIS

Gets device roles from the NinjaOne API.

## SYNTAX

```powershell
Get-NinjaOneRoles [<CommonParameters>]
```

## DESCRIPTION

Retrieves device roles from the NinjaOne v2 API.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-NinjaOneRoles
```

Gets all device roles.
