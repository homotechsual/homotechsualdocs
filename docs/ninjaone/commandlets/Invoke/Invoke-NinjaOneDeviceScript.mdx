## SYNOPSIS

Runs a script or built-in action against the given device.

## SYNTAX

```powershell
Invoke-NinjaOneDeviceScript [-deviceId] <String[]> [-type] <Object> [[-scriptId] <Int32>]
 [[-actionId] <String>] [[-parameters] <String>] [[-runAs] <String>] [<CommonParameters>]
```

## DESCRIPTION

Runs a script or built-in action against a single device using the NinjaOne v2 API.
