## SYNOPSIS

Sets location information, like name, address, description etc.

## SYNTAX

```powershell
Set-NinjaOneLocation [-organisationId] <Int32> [-locationId] <Int32> [-locationInformation] <Object> [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

Sets location information using the NinjaOne v2 API.
