## SYNOPSIS

Sends a request to the NinjaOne API.

## SYNTAX

```powershell
Invoke-NinjaOneRequest [-method] <String> [-uri] <String> [[-body] <String>] [-raw] [-parseDateTime]
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Wrapper function to send web requests to the NinjaOne API.

## EXAMPLES

### EXAMPLE 1

```powershell
Invoke-NinjaOneRequest -method 'GET' -uri 'https://eu.ninjarmm.com/v2/activities'
```


Make a GET request to the activities resource.

### EXAMPLE 2

```powershell
Invoke-NinjaOneRequest -method 'GET' -uri 'https://eu.ninjarmm.com/v2/activities' -parseDateTime
```


Parse ISO 8601 and Unix epoch values in the response to [DateTime].

### EXAMPLE 3

```powershell
Invoke-NinjaOneRequest -method 'GET' -uri 'https://eu.ninjarmm.com/v2/activities' -parseDateTime
```


Make a GET request and convert ISO 8601 or Unix epoch values to [DateTime].
