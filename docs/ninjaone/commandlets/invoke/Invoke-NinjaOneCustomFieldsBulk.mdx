## SYNOPSIS

Performs bulk operations on custom fields.

## SYNTAX

```powershell
Invoke-NinjaOneCustomFieldsBulk [-Operation] <String> [-CustomFields <Object>] [-FieldNames <String[]>]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

Performs bulk create, update, or delete operations on custom fields via the NinjaOne v2 API.

## EXAMPLES

### EXAMPLE 1

```powershell
Invoke-NinjaOneCustomFieldsBulk -Operation 'create' -CustomFields @( @\{ name = 'Field1' \}, @\{ name = 'Field2' \} )```


Creates multiple custom fields in a single request.
