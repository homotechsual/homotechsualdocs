## SYNOPSIS

Creates a new policy using the NinjaOne API.

## SYNTAX

```powershell
New-NinjaOnePolicy [-mode] <String> [-policy] <Object> [[-templatePolicyId] <Int32>] [-show] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

Create a new policy using the NinjaOne v2 API.
