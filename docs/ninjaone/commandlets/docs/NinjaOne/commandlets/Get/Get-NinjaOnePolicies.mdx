## SYNOPSIS

Gets policies from the NinjaOne API.

## SYNTAX

```powershell
Get-NinjaOnePolicies [<CommonParameters>]
```

## DESCRIPTION

Retrieves policies from the NinjaOne v2 API.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-NinjaOnePolicies
```

Gets all policies.
