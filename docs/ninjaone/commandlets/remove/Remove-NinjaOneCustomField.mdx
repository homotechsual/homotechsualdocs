## SYNOPSIS

Deletes a custom field by field name.

## SYNTAX

```powershell
Remove-NinjaOneCustomField [-FieldName] <String> [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION

Deletes an existing custom field identified by its unique field name via the NinjaOne v2 API.

## EXAMPLES

### EXAMPLE 1

```powershell
