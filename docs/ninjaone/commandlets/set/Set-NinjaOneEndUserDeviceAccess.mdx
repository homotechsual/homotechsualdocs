## SYNOPSIS

Updates end user device access.

## SYNTAX

```powershell
Set-NinjaOneEndUserDeviceAccess [-Id] <Int32> [-DeviceAccess] <Object> [-ProgressAction <ActionPreference>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

Updates device access settings for an end user using the NinjaOne v2 API.

## EXAMPLES

### EXAMPLE 1

```powershell
Set-NinjaOneEndUserDeviceAccess -Id 101 -DeviceAccess @{ canRemote = $true }
```


Updates device access settings for end user 101.
