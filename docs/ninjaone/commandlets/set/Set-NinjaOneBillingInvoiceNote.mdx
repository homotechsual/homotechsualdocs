## SYNOPSIS

Updates a billing invoice note.

## SYNTAX

```powershell
Set-NinjaOneBillingInvoiceNote [-id] <Int32> [-note] <Object> [-ProgressAction <ActionPreference>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

Updates the note for a billing invoice using the NinjaOne v2 API.

## EXAMPLES

### EXAMPLE 1

```powershell
Set-NinjaOneBillingInvoiceNote -id 1 -note @{ note = 'Customer confirmed billing details' }
```


Updates the note on billing invoice 1.
