## SYNOPSIS

Sets the value of the specified device custom fields.

## SYNTAX

```powershell
Set-NinjaOneDeviceCustomFields [-deviceId] <Int32> [-deviceCustomFields] <Object> [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION

Sets the value of the specified device custom fields using the NinjaOne v2 API.

## EXAMPLES

### EXAMPLE 1

```powershell
Set-NinjaOneDeviceCustomFields -deviceId 1 -customFields @{ CustomField1 = 'Value1'; CustomField2 = 'Value2' }
```

Set `CustomField1` to `Value1` and `CustomField2` to `Value2` respectively for the device with id 1.
