## SYNOPSIS

Gets users from the NinjaOne API.

## SYNTAX

### Default (Default)

```powershell
Get-NinjaOneUsers [[-userType] <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

### Organisation

```powershell
Get-NinjaOneUsers [-organisationId] <Int32> [[-userType] <String>] [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

## DESCRIPTION

Retrieves users from the NinjaOne v2 API.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-NinjaOneUsers
```

Gets all users.

### EXAMPLE 2

```powershell
Get-NinjaOneUsers -userType TECHNICIAN
```

Gets all technicians (users with the TECHNICIAN user type).

### EXAMPLE 3

```powershell
Get-NinjaOneUsers -organisationId 1
```

Gets all users for the organisation with id 1 (only works for users with the END_USER user type).
