## SYNOPSIS

Gets device windows services from the NinjaOne API.

## SYNTAX

```powershell
Get-NinjaOneDeviceWindowsServices [-deviceId] <Int32> [[-name] <String>] [[-state] <String>]
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Retrieves device windows services from the NinjaOne v2 API.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-NinjaOneDeviceWindowsServices -deviceId 1
```

Gets all windows services for the device with id 1.

### EXAMPLE 2

```powershell
Get-NinjaOneDeviceWindowsServices -deviceId 1 -name 'NinjaRMM Agent'
```

Gets all windows services for the device with id 1 that match the name 'NinjaRMM Agent'.

### EXAMPLE 3

```powershell
Get-NinjaOneDeviceWindowsServices -deviceId 1 -state 'RUNNING'
```

Gets all windows services for the device with id 1 that are in the 'RUNNING' state.
