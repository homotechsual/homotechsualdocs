## SYNOPSIS

Sends a message using the Microsoft Graph API.

## SYNTAX

```powershell
Send-MSGraphMail -Mailbox <String> -MessageID <String[]> [-Folder <String>] [<CommonParameters>]
```

## DESCRIPTION

Sends a previously created message using the Microsoft Graph API.
The message can be sent from a folder, or from the root of the mailbox.
