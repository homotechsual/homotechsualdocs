## SYNOPSIS

Removes an email from a mailbox using the Microsoft Graph API.

## SYNTAX

```powershell
Remove-MSGraphMail -Mailbox <String> -MessageID <String[]> [-Folder <String>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION

Deletes an email from a mailbox using the Microsoft Graph API.
The mailbox can be specified using a UPN or a mailbox ID.
The mailbox can be filtered using a folder ID.
