## SYNOPSIS

Moves emails from a mailbox using the Microsoft Graph API.

## SYNTAX

```powershell
Move-MSGraphMail -Mailbox <String> -MessageID <String[]> [-Folder <String>] [-Destination <String>]
 [<CommonParameters>]
```

## DESCRIPTION

Moves emails between folders in a mailbox using the Microsoft Graph API.
The mailbox can be specified using a UPN or a mailbox ID.
The mailbox can be filtered using a folder ID.
